# Practice react & D3

## Project
  練習從頭建立React & Redux & D3, 改用parcel 打包React

## script
  - 安裝package
    `yarn`
  - 啟動project
   `yarn start`

## 使用
  - [x] React
  - [x] React-router

## 步驟
  1. 新增專案
  2. 加入 React component
    `yarn add -SE react react-dom`
  3. 加入 BABEL
    `yarn add -DE babel-preset-react`
  4. 添加 Parcel
    `yarn add -DE parcel-bundler`
  5. 添增 react-router
    `yarn add -SE react-router-dom `
